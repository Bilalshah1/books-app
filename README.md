## Getting Started


## Environment Setup

Create a `.env` file in the root and add your [Google Books API key](https://console.cloud.google.com/):

```
EXPO_PUBLIC_GOOGLE_BOOKS_API_KEY='Your key here'
```

## Requirements
- Node.js
- Expo Go app (iOS/Android)
